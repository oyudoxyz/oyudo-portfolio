Redefining Content Consumption with AI

Risten is a platform aimed at revolutionizing how users engage with content by addressing their unique challenges and preferences.

Overview

This project delves into understanding user needs through surveys, crafting user personas, mapping workflows, and proposing solutions that resonate with modern content consumption habits.

Role

Product Designer

Tools

Figma, Google forms, Tableu

Focus Areas

User Research, Personas, User Flows, Site Maps, Visual Design

Problem Statement

Current platforms often fail to cater to users' evolving content consumption needs. Issues such as limited engagement options, lack of personalization, and user fatigue with complex interfaces create friction in content discovery and interaction. Risten seeks to address these challenges by building a more intuitive, personalized, and user-centered platform.

Research and Insights

To understand the users' challenges and preferences, we conducted a comprehensive survey with 59 participants.

Key Insights

Frequency of Reading Articles or Newsletters: Users demonstrated a wide variety of habits, from daily readers to those who consume content sporadically.

Preferred Ways to Engage with Content: Respondents showed strong preferences for formats like articles and podcasts, emphasizing the need for diverse engagement methods.

These insights informed the development of personas, user journeys, and the initial features for Risten.

Empathy: User Personas

Based on survey results and additional research, we developed four core personas that represent our key user groups.

Casual Reader Clara: Values quick, skimmable content and easy access.

News-Focused Noah: Seeks detailed and reliable news updates.

Multitasker Morgan: Prefers podcasts and audiobooks for hands-free engagement.

Tech-Savvy Tim: Appreciates personalization and a clean, intuitive interface.

Definition and Ideation

Using the insights, we created comprehensive user flows and site maps to organize platform features intuitively.

Key Features

Multi-format content delivery (articles, podcasts, etc.)
Personalized recommendations based on user preferences
Simplified interface with accessibility features
Content discovery tools
User engagement analytics
Cross-platform synchronization
